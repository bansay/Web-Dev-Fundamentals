But quickly, why the HTML5 Boilerplate is a GREAT jumpstart for folks wanting to dive into code:

The boilerplate became popular for a few reasons:

  • Provides fast jumping off point when you have deadlines for projects, and you want all the fixins:
    • reset css
      • includes normalize
      • browser compatibility from the get go
    • working root directory structure to build on
      • jquery plugged in and ready
      • all your scripts and css files are already added and ready to go
    • save time from dilly dally tasks that become annoying after repetition. save time and get to writing code faster!
